# Aplicativos
Vários deles terão 2 fotos da mesma página demonstrando a variação que implementei do Modo noturno, utilizando regras CSS e um pouco de JavaScript para que as Configurações do telefone permita o usuário de definir se deseja utilizar o Modo noturno ou não.

## Apostas Diamond (Paródia da Blaze)
#### Aplicativo inspirado na Blaze, contendo 3 páginas, cada uma criada para um tipo diferente de modalidade de aposta, sendo elas: **Crash**, **Double** e **Mines**.
#### Todos as 3 modalidades de jogos foram programados por mim, com lógicas para definir a taxa de lucro do usuário e também a probabilidade dele ganhar ou perder nos jogos.

## Relógio
#### Aplicativo que provê quase tudo que o aplicativo provê na vida real. Construí para ele:
- Alarmes: Podem ser criados com horários específicos e nomes para os alarmes, além de poder apagar ou desativar os alarmes.
- Cronômetro: Permite iniciar o cronômetro, indicar quando uma volta foi concluída e permitir visualizar o tempo decorrido de todas as voltas que foram marcadas pelo usuário.
- Temporizador: É possível indicar pra daqui quanto tempo o telefone deverá disparar um alerta, através da indicação das horas, minutos e/ou segundos de espera que o aplicativo deve respeitar.

## Telefone
#### Aplicativo que trás funcionalidades básicas relacionadas a Ligações como: Discador, Histórico de Ligações e Lista de Contatos.
- O **Discador** permite realizar ligações anônimas ao utilizar o símbolo * ou # no início da digitação.
- O **Histórico de Ligações** mostra as ligações que foram recebidas, efetuadas, não atendidas, etc, além de permitir o usuário de apagar históricos específicos.
- A **Lista de Contatos** é uma versão resumida e enxuta do aplicativo **Contatos**.

## U-beer (Paródia do Uber)
#### Aplicativo que foi construído para representar um aplicativo tipo Uber, porém extremamente simplificado, onde é possível usuários se cadastrarem rapidamente como Motoristas, e outros como Passageiros.
#### Os motoristas receberiam alertas quando um Passageiro fosse cadastrado no sistema, e os motoristas poderiam aceitar as Corridas ou ignorá-las.
#### Os valores eram calculados automaticamente baseado na distância do trajeto.
